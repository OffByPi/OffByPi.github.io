## About OffByPi

OffByPi is a personal developer hub curated by Dani (dani84bs). It is designed to document real-world systems engineering problems, software architecture decisions, and development workflows.

### Stack & Tools

My daily development environment relies on robust, keyboard-driven, and reproducible tools:
- **OS:** macOS & Debian Linux
- **Environment:** chezmoi and Fish Shell
- **Editor:** Obsidian (configured as a local Zettelkasten with Vim keybindings)
- **Engine:** Built with Quartz and managed via Git

If you wish to get in touch or explore my open-source work, visit my [GitHub](https://github.com/dani84bs).
